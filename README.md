## 简介
一款基于iOS的学习规划/时间安排/笔记管理/在线测验/学习辅助App  
Swift 4开发Carthage第三方管理



## 已实现的功能
- 功能1:Todo事项增加、管理、提醒
- 功能2:笔记增加、管理、搜索
- 功能3:在线测验
- 功能4:学习曲线生成
- 功能5:学习打卡  
- 功能6:登陆、注册、注销、找回密码 
- 功能7:头像上传、更换
- 功能8:支持3D Touch
