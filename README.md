## 简介
一款基于iOS的学习规划/时间安排/笔记管理/在线测验/学习辅助记录App  
开发语言：Swift 4  
第三方管理：Carthage  
如果觉得有用的，欢迎点右上角Star收藏，感谢～



## 已实现的功能
- 功能1:Todo事项增加、管理、提醒、可设置重复提醒日期
- 功能2:笔记增加、管理、搜索
- 功能3:在线测验
- 功能4:学习曲线生成
- 功能5:学习打卡  
- 功能6:登陆、注册、注销、找回密码 
- 功能7:头像上传、更换
- 功能8:支持3D Touch
